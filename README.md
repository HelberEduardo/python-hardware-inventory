# Hardware Inventory Script

Script em Python para coleta automatizada de especificações de hardware e rede.

## Tecnologias
- Python 3.x
- Biblioteca Psutil (Monitoramento de sistema)

## Como rodar
1. Clone o repositório
2. Instale as dependências: `pip install -r requirements.txt`
3. Execute: `python inventario.py`

## Resultado
O script gera um arquivo `Info_System.txt` com o relatório completo da máquina.
